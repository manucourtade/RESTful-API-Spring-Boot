RESTful API con Spring Boot y MySQL
Este proyecto es una API RESTful construida con Spring Boot que demuestra el manejo básico del framework. La arquitectura sigue la convención de capas separadas en paquetes — Controller, Service y Model — donde cada capa tiene una responsabilidad específica y se comunica con la siguiente.
Para la persistencia de datos se integró Spring Data JPA con MySQL, lo que permite guardar, actualizar y eliminar productos de forma real. El manejo de errores se implementó con ResponseEntity devolviendo los códigos HTTP correctos según cada situación.
Tecnologías: Java 21 · Spring Boot 4.1 · Spring Data JPA · Hibernate · MySQL · Maven
A futuro: validaciones, DTOs, manejo global de excepciones y autenticación con JWT.

RESTful API with Spring Boot and MySQL
A RESTful API built with Spring Boot showcasing the fundamentals of the framework. The project follows a layered architecture with separated packages — Controller, Service, and Model — where each layer has a single responsibility and communicates with the next.
Data persistence is handled through Spring Data JPA with MySQL, enabling full CRUD operations. Error handling is implemented using ResponseEntity with proper HTTP status codes.
Tech stack: Java 21 · Spring Boot 4.1 · Spring Data JPA · Hibernate · MySQL · Maven
Coming soon: validations, DTOs, global exception handling, and JWT authentication.
